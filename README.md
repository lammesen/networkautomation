# Network Automation Platform

This repository contains a full-stack network automation platform built with FastAPI, Celery, Nornir, and a React frontend.

## Structure

```
backend/   # FastAPI app, SQLAlchemy models, Celery tasks
frontend/  # React + TypeScript SPA
deploy/    # Docker Compose stack and runtime images
docs/      # Architecture, API, and operations references
```

Refer to `docs/operations.md` for setup instructions.
